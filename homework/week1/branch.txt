What is a so called “branch” in Git terms and why would you use it. “Een branch in Git is simpelweg een lichtgewicht verplaatsbare verwijzing naar een van deze commits. De standaard branch-naam in Git is master. Als je initieel commits maakt, dan krijg je een master-branch die wijst naar de laatste commit die je gemaakt hebt. Iedere keer als je commit, beweegt het automatisch vooruit.” (https://git-scm.com/book/nl/v1/Branchen-in-Git-Wat-is-een-branch)
